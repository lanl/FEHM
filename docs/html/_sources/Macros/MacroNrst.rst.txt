========
``nrst``
========

Stop NR iterations on variable changes (not equation tolerance). 

* Group 1 - VARD, STOPC

+----------------+-----------+-------------------------------------+
| Input Variable | Format    | Description                         |
+================+===========+=====================================+
| VARD           | character | Variable to check.                  |
+----------------+-----------+-------------------------------------+
| STOPC          | real      | Value to check on variable change.  |
+----------------+-----------+-------------------------------------+
