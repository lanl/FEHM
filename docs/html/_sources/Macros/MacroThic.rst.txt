========
``thic``
========

Input for variable thickness for two-dimensional problems.

* Group 1 - JA, JB, JC, THIC

+----------------+--------+-------------------------------------------------------------------------+
| Input Variable | Format | Description                                                             |
+================+========+=========================================================================+
| THIC           | real   | Thickness of the model domain in the third dimension (m). Default is 1. |
+----------------+--------+-------------------------------------------------------------------------+
