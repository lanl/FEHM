========
``flo3``
========

Alternate format for flow data for a seepage face.

+----------------+---------+---------------------------------------------------------------+
| Input Variable | Format  | Description                                                   |
+================+=========+===============================================================+
| PFLOW          | real    | Ambient pressure (MPa) or flow area (m2)                      |
+----------------+---------+---------------------------------------------------------------+
| ESK            | real    | Not used                                                      |
+----------------+---------+---------------------------------------------------------------+
| AIPED          | real    | Same as above for AIPED under keyword "flow"                  |
+----------------+---------+---------------------------------------------------------------+
| KA             | integer | Flag to indicate seepage face (-3) or unit gradient flux (-2) |
+----------------+---------+---------------------------------------------------------------+
